## 🎵 Apple Music API Integration

The app now features a professional-grade integration with the Apple Music (MusicKit) API.

*   **Built-in Credentials**: Bundled at build time from environment variables (not committed to the repo).
*   **Automatic Auth**: Generates ES256-signed JWT tokens with a 180-day lifetime and automatic refresh.
*   **ISRC Priority**: The search cascade now follows this priority:
    1.  **ISRC Lookup**: 100% accurate match via Apple Music API.
    2.  **MusicBrainz (Local)**: Fast offline search for bulk data.
    3.  **Apple Music Catalog**: High-quality online text search.
    4.  **iTunes API**: Final fallback for fuzzy matching.

---

## 📂 Enhanced Export Formats

Users can now choose their destination format upon saving:

1.  **Last.fm CSV**: The standard 6-column format for Last.fm scrobblers.
2.  **ListenBrainz JSON**: Native JSON format with **Unix Epoch timestamps** for direct import to ListenBrainz.org.
3.  **Universal CSV**: A "lossless" format that preserves every single original column alongside the matched artist info.
4.  **Spotify CSV**: Optimized for third-party Spotify history importers (includes duration in milliseconds).
